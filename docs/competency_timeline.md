## Хронологический трек (сводный)

### ~1998 — Подготовительное отделение ОмГУ
- **Математика:** алгебра, геометрия — первое глубокое понимание (🟢 для уровня)
- **Английский:** грамматический фундамент с нуля
- **ПК:** первый опыт — Windows 95, Word, Excel, Netscape Navigator

### 1999–2003 — ОмГУ, экономический факультет
- **Математика:** высшая математика (2 сем.), теорвер, матстат, теория игр, линейное программирование, экономико-математические методы
- **Английский:** неформальное освоение — фильмы, игры, музыка на языке оригинала; рост словарного запаса
- **ПК / веб:** HTML, CSS, PHP, JS — через окружение (друг-веб-разработчик)
- **ПК / инструменты:** Far Manager, файловые системы
- **БД:** первое знакомство с концепцией СУБД (факультатив)
- **Программирование:** знакомство с ООП на Java (факультатив), основы криптографии
- **Excel:** продвинутый уровень

### 2004–2006 — Первая работа (корпоративный учёт, аналог 1С)
- **SQL:** первый боевой опыт с реальными БД
- **FoxPro:** программирование отчётов под заказчика
- **БД:** reverse engineering структуры без документации
- **Сети:** настройка локальных сетей, подбор оборудования

### 2003–2006 — Аспирантура ОмГУ
- **Английский:** чтение и перевод научных статей, значительный рост словаря
- **Математика:** углублённый математический анализ, статистика в исследованиях
- **Данные:** статистические программные комплексы, международные БД (МВФ, Мировой банк)

### 2006–2010 — Экономист (строительство)
- **Excel:** экспертный уровень — прогнозирование, факторный анализ, экстраполяция, сложные модели
- **Аналитика:** бюджетирование, управленческий учёт затрат, проектный анализ
- **Визуализация:** отчёты и дашборды для руководства

### 2010–2018 — Инженер (строительство, снабжение) + ОмГТУ (заочно)
- **CAD / моделирование:** AutoCAD и аналоги, 3D-моделирование, МКЭ (конечноэлементный анализ), численное моделирование физических процессов
- **Технические дисциплины:** физика, сопромат
- **БД:** Access — проектирование и ведение собственных БД
- **SQL:** углублённое изучение, детализированный учёт номенклатуры
- **Английский:** периодическая пресса на языке оригинала (систематически)

### 2018–2021 — Поворот: приватность → Linux → сети → облака
- **Linux:** первое погружение с нуля, самостоятельно; первые self-hosted серверы
- **Сети:** DNS, домены, хостинг, веб-инфраструктура; VPN, proxy, туннели, шифрование трафика; Tor, p2p, L2TP; сетевая безопасность
- **Веб:** WordPress, статические сайты, HTML/CSS (переосмысление с практикой)
- **Облака:** AWS, Azure — VPS, виртуализация, масштабирование, облачные сервисы
- **Удалённый доступ:** RDP, VNC и аналоги

### 2022 — PHP → Python
- PHP: начало изучения (недолго)
- **Python:** старт через курс "Аналитик данных" (Яндекс, 350 ч.)
- Pandas, NumPy, основы визуализации, SQL в связке с Python

### 2023 — Python на практике + Linux окончательно
- **Python:** применение на работе; осознание нехватки базы → углублённое изучение
- Проект **nier** — MLP с нуля на NumPy (*личная инициатива, не в рамках курса*)
- **Linux:** полный переход (работа + дом)
- **Tooling:** git, IDE (углублённо)
- Начало системного изучения **Computer Science**

### Конец 2023 – 2024 — Системное CS + многоязычность
- **CS:** ООП (углублённо), алгоритмы и структуры данных, компьютерные сети (углублённо), архитектура компьютера
- **Языки программирования:** LeetCode medium — Python, PHP, Java, Rust; экспресс-курсы: Go, JavaScript, C++; сравнительный обзор дизайна языков
- **Rust:** углублённое изучение (осень 2024)
- **Java:** изучение, вступительные испытания на Java Developer (конец 2024)
- **Пет-проекты:** CLI-утилиты (криптография), тренажёр печати, Telegram-боты, сайты
- **AI-ассистенты:** систематическое использование в обучении и кодинге
- **Периодика:** специализированная техническая литература (систематически)

### 2025 — MLOps + парадигмы + платформы
- **DevOps:** Docker, GitHub Actions, CI/CD
- **API:** FastAPI
- **Парадигмы:** сдвиг ООП → функциональное программирование
- **Агентское программирование:** начало практики
- **Hugo / Congo:** статические сайты, шаблонизация
- Курс "Управление данными, бизнес-анализ" (Уфа, 80 ч.)
- **Вектор:** RAG, MCP, LLM в продакшн — *направление развития, не текущий навык*

### 2026 — настоящее
- **SigilGate:** сетевая инфраструктура, DPI-обход, Docker-сервисы, Python-бэкенд, Telegram-бот
- **info-tech-io:** образовательная платформа, multi-repo архитектура, Hugo-шаблоны
- **a1exma.ru:** этот сайт — Hugo, GitHub Actions, multi-branch deployment
